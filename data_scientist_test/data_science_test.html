Definindo uma Métrica de Desempenho¶

difcil medir a qualidade de um modelo dado sem quantificar seu desempenho durante o treinamento e teste. Isso geralmente feito utilizando algum tipo de mtrica de desempenho, atravs do clculo de algum tipo de erro, qualidade de ajuste, ou qualquer outra medida til.

Os valores para R2 tm um alcance de 0 a 1, que captura a porcentagem da correlao ao quadrado entre a estimativa e o valor atual da varivel alvo. Um modelo R2 de valor 0 sempre falha ao estimar a varivel alvo, enquanto que um modelo R2 de valor 1, estima perfeitamente a varivel alvo.

Na implementao abaixo criei a funo performance_metric, onde utilizei o r2_score do sklearn.metris com intuito de realizar um calculo de desempenho entre y_true e y_predict, e tambm atribui a pontuao do desempenho a varivel score.

In [22]:
def performance_metric(y_true, y_predict):
    """Calcular e retornar a pontuao de desempenho entre
    valores reais e estimados baseado na mtrica escolhida."""
    
    # Importar o 'r2_score'
    from sklearn.metrics import r2_score
    
    # TODO: Calcular a pontuao de desempenho entre 'y_true' e 'y_predict'
    score = r2_score(y_true, y_predict)
    
    # Devolver a pontuao
    return score

Ajustando o modelo¶

Na implementao abaixo acontece o treinamento do modelo utilizando o algoritimo de rvore de deciso. Como garantia de estar utilizando um modelo otimizado treinei o modelo utilizando busca em matriz com intuito de otimizar o parmeto de max_depth (profundidade maxima) para a rvore de deciso, esse parametro significa o entendimento sobre o nmero de perguntas que o algoritimo deve fazer sobre os dados antes de uma estiatica. rvores de deciso so otimos algoritimos para se trabalhar aprendizagem supervisionada.

  • Para esse modelo utilizei DecisionTreeRegressor do sklearn.tree para gerar um objeto regressor da rvore de deciso.
  • Gerei um dicionrio em max_depth com valores de 1 a 10.
  • Utilizei make_scorer do sklearn.metrics para gerar um objeto de funo de pontuao.
  • Tambm utilizei GridSearchCV do sklearn.grid para gerar um objeto de busca em matriz.
In [28]:
from sklearn.tree import DecisionTreeRegressor
from sklearn.grid_search import GridSearchCV
from sklearn.metrics import make_scorer

def fit_model(X, y):
    """Desempenhar busca em matriz sobre o parmetro 'max_depth' para uma
    rvore de deciso de regresso treinada nos dados de entrada [x, y]"""
    
    # Gerar conjuntos de validao-cruzada para o treinamento de dados
    cv_sets = ShuffleSplit(X.shape[0], n_iter = 10, test_size = 0.20, random_state = 0)
    
    # TODO: Gerar uma rvore de deciso de regresso de objeto
    regressor = DecisionTreeRegressor(random_state=0)
    
    # TODO: Gerar um dicionrio para o parmetro 'max_depth' com um alcance de 1 a 10
    params = {'max_depth': range(1,11)}
    
    # TODO: Transformar 'performance_metric' em uma funo de pontuao utilizando 'make_scorer' 
    scoring_fnc = make_scorer(performance_metric)
    
     # TODO: Gerar o objeto de busca em matriz
    grid = GridSearchCV(regressor, params, scoring_fnc, cv=cv_sets, verbose=0)
    
    # Ajustar o objeto de busca em matriz com os dados para calcular o modelo timo
    grid = grid.fit(X, y)
    
    # Devolver o modelo timo depois de realizar o ajuste dos dados
    return grid.best_estimator_
